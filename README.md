# Expense-tracker
A Python-based Expense Tracker that uses file handling to store, manage, and analyze daily expenses through a simple command-line interface.
Developed an Expense Tracker application using Python and file handling concepts to manage daily expenses efficiently. The project allows users to add, view, search, and delete expense records. All expense data is stored in a text file, ensuring data persistence even after the program is closed. The application demonstrates the use of Python fundamentals, file operations (read/write/append), functions, loops, and conditional statements to provide a simple and user-friendly expense management system.
Key Features:
Add new expenses with details
View all recorded expenses
Search expenses by category or date
Delete expense records
Store and retrieve data using file handling
Simple command-line interface
Technologies Used:
Python
File Handling (Text Files)
Functions and Data Structures
